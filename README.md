# 东南大学博士学位论文latex模板

本项目在[seuthesix](https://github.com/zhimengfan1990/seuthesix)的基础上进行微调，通过了2025届工学博士毕业论文的格式审查。请将项目克隆到本地，接着导入至overleaf或texpage等网站以开始学位论文的写作。

# Tips

1. 尽可能插入pdf格式的图片；切忌使用png等格式图片，会严重影响编译效率！！！
2. 主文件为main.tex。建议在分章写作时，注释掉无关章节对应的```\include```指令以缩短编译时间。
3. 在abstract_cn.tex文件中，可以选择生成大封面（\makebigcover，用于论文打印）还是小封面（\makecover）。
4. 祝诸君一切顺利，早日毕业。
